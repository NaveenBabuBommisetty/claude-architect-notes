## How to Use This Tutorial Series

1. Read each domain file in order from `01` to `07`
2. After each section, try to answer the concept check questions from memory
3. Run the code examples (Python/JSON) yourself
4. Review the anti-patterns file separately — these are exam traps
5. Do all 60 practice questions in file `07` under timed conditions (60 minutes)
